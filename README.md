# Schedule Manager API

1. Clone repositry

2. run:
```
npm install
yarn install 
```
3. rename .env.template file to .env

4. fill with your own env variables

5. run DB:
```
docker-compose up -d
```

6. start development mode
```
npm run start:dev
yarn start:dev
```# schedule_manager
